キャップストーン・プレゼンテーション提出詳細

FDEは以下を提出する必要があります：
- PowerPoint プレゼンテーション（PPT）
- プロジェクトの GitHub リポジトリリンク
- ソースコード（ZIPファイル）

キャップストーン・プレゼンテーション
各参加者は、別々のミーティングルームで、パネリストの前にてプロジェクト発表を行います。
プレゼンテーション内容：
- プロジェクトのプレゼンテーション
- プロジェクトのライブデモ
- 質疑応答セッション
※ プレゼンテーションの順番はランダムに決定されます。

締切前にすべての提出が完了していること、またライブデモに向けてプロジェクトが万全に準備されていることをご確認ください。

Capstone Presentation Submission Details
FDE's are required to submit the following:
PowerPoint Presentation (PPT)
GitHub repository link of the project
Source code (ZIP file)

The Capstone Presentation
Each Participant will present their project in front of the panelists in a separate meeting room.
The presentation will include:
Project presentation
Live demo of the project
Question & Answer session
Please note that presentation order will be decided randomly.
Kindly ensure that all submissions are completed before the deadline and that your project is fully prepared for the live demonstration.
